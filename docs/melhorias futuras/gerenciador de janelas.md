1. Operações básicas de janela

O Windows permite:

Abrir uma janela.
Fechar uma janela.
Minimizar para a barra de tarefas.
Maximizar para ocupar a tela.
Restaurar ao tamanho anterior.
Mover pela barra de título.
Redimensionar pelas bordas e cantos.
Abrir o menu da janela.
Colocar uma janela em primeiro plano.
Alternar o foco entre janelas.
Manter várias janelas abertas simultaneamente.
Controlar qual janela fica acima das outras.
Organizar janelas sobrepostas.
Exibir caixas de diálogo pertencentes a uma janela.
Manter janelas-filhas associadas ao aplicativo principal.

Os atalhos tradicionais do menu da janela incluem Alt + Espaço, que permite restaurar, mover, redimensionar, minimizar, maximizar ou fechar a janela ativa.

2. Alternância entre janelas

O gerenciador permite encontrar e alternar rapidamente entre aplicativos:

Alt + Tab: alternar entre janelas abertas.
Alt + Shift + Tab: alternar no sentido contrário.
Windows + Tab: abrir a Visão de Tarefas.
Clicar no ícone do aplicativo na barra de tarefas.
Alternar entre janelas do mesmo aplicativo.
Visualizar miniaturas ao passar o mouse sobre a barra de tarefas.
Fechar uma janela diretamente pela miniatura.
Restaurar uma janela minimizada.
Alternar entre janelas usando toque ou trackpad.
Mostrar grupos de janelas organizadas pelo Snap.

As miniaturas exibidas na barra de tarefas e na troca de janelas são representações dinâmicas produzidas pelo DWM, e não apenas imagens estáticas.

3. Snap de janelas

O Snap organiza automaticamente as janelas na tela:

Posicionar na metade esquerda.
Posicionar na metade direita.
Posicionar no canto superior esquerdo.
Posicionar no canto superior direito.
Posicionar no canto inferior esquerdo.
Posicionar no canto inferior direito.
Maximizar arrastando para o topo.
Ajustar janelas pelo mouse, toque ou teclado.
Redimensionar simultaneamente janelas encaixadas.
Preencher automaticamente o espaço restante.
Mostrar sugestões de outras janelas abertas.
Organizar duas, três ou quatro janelas.
Usar configurações diferentes conforme o tamanho do monitor.
Atalhos de Snap
Windows + esquerda: encaixar à esquerda.
Windows + direita: encaixar à direita.
Windows + cima: maximizar ou mover para a parte superior.
Windows + baixo: restaurar, minimizar ou mover para baixo.
Windows + Z: abrir os layouts do Snap no Windows 11.
4. Layouts do Snap

No Windows 11, ao passar o mouse sobre o botão de maximizar ou pressionar Windows + Z, são exibidos layouts predefinidos.

Eles permitem:

Dividir a tela em duas partes iguais.
Criar uma área maior e outra menor.
Organizar três colunas.
Criar uma janela principal e duas secundárias.
Dividir a tela em quatro áreas.
Escolher uma posição dentro do layout.
Preencher as posições restantes com o Snap Assist.
Exibir layouts diferentes conforme resolução e tamanho da tela.
Abrir a barra de layouts arrastando uma janela para o topo da tela.
5. Snap Assist

Depois de encaixar uma janela, o Snap Assist mostra as outras janelas abertas para preencher o espaço disponível.

Suas funções incluem:

Mostrar miniaturas das janelas disponíveis.
Escolher rapidamente a segunda janela.
Preencher todas as posições do layout.
Ignorar uma posição.
Substituir uma janela do grupo.
Redimensionar janelas encaixadas juntas.
Evitar espaços vazios entre as janelas.
Sugerir aplicativos usados recentemente.
6. Grupos de Snap

Quando várias janelas são organizadas juntas, o Windows pode criar um grupo de Snap.

Isso permite:

Alternar para todo o conjunto de janelas.
Restaurar a disposição anterior.
Mostrar o grupo na barra de tarefas.
Mostrar o grupo no Alt + Tab.
Mostrar o grupo na Visão de Tarefas.
Restaurar apenas uma janela do grupo.
Retornar ao ambiente de trabalho depois de abrir outro aplicativo.

Os grupos são formados automaticamente quando múltiplas janelas são encaixadas no mesmo layout.

Visualizar todas as janelas abertas.

Fechar janelas.

Criar áreas de trabalho virtuais.
Alternar entre áreas de trabalho.

As áreas de trabalho virtuais permitem separar conjuntos de janelas. Por exemplo:

Uma área para programação.
Uma para navegação.
Uma para jogos.
Uma para comunicação.
Uma para trabalho ou estudo.

As funções incluem:

Criar uma nova área de trabalho.
Alternar entre áreas.
Fechar uma área.
Mover uma janela de uma área para outra.
Exibir uma janela em várias áreas, dependendo das opções disponíveis.
Separar aplicativos por atividade.
Escolher se a barra de tarefas mostra apenas as janelas da área atual ou de todas as áreas.
Escolher se Alt + Tab mostra janelas da área atual ou de todas.
Atalhos

Windows + Ctrl + esquerda: ir para a área anterior.
Windows + Ctrl + direita: ir para a próxima área.
Windows + Ctrl + F4: fechar a área atual.
Windows + Tab: abrir a Visão de Tarefas.

Ao fechar uma área de trabalho virtual, os aplicativos não são necessariamente encerrados; suas janelas são movidas para outra área disponível.

17. Desktop Window Manager — dwm.exe

Internamente, o DWM é responsável pela composição da área de trabalho.

Em vez de cada programa desenhar diretamente na tela:

Cada janela desenha seu conteúdo em uma superfície separada na memória.
O DWM recebe essas superfícies.
Ele combina as janelas, área de trabalho, barra de tarefas, sombras e efeitos.
A imagem final é enviada ao monitor.

Essa arquitetura reduz artefatos gráficos, facilita animações e permite que uma janela coberta continue com seu conteúdo disponível para composição.

18. Funções internas do DWM

Entre as principais funções do DWM estão:

Compor todas as janelas na imagem final da área de trabalho.
Manter superfícies separadas para os aplicativos.
Controlar animações de janela.
Renderizar sombras e transparências.
Produzir desfoque de fundo.
Aplicar efeitos às áreas não clientes das janelas.
Produzir miniaturas em tempo real.
Apoiar o Alt + Tab.
Apoiar miniaturas da barra de tarefas.
Auxiliar no dimensionamento de aplicativos antigos em telas com DPI elevado.
Coordenar a apresentação dos quadros.
Eliminar rastros visuais ao mover janelas.
Combinar conteúdo vindo de diferentes aplicativos e monitores.
19. Consumo de GPU e memória

O processo dwm.exe pode utilizar:

GPU para composição.
Memória de vídeo.
Memória RAM compartilhada.
CPU em determinadas operações.
Recursos adicionais quando existem muitos monitores.
Mais memória quando existem muitas janelas.
Mais recursos com resoluções elevadas, HDR, transparência ou múltiplas telas.

Como ele é um componente central da interface gráfica, normalmente não deve ser encerrado manualmente.

20. Escala e DPI

O gerenciador de janelas também participa da adaptação para diferentes escalas:

100%, 125%, 150%, 200% ou outras.
Monitores com escalas diferentes.
Aplicativos compatíveis com DPI por monitor.
Redimensionamento ao mover uma janela entre telas.
Escala de aplicativos antigos.
Textos, bordas e controles adaptados à densidade do monitor.

O DWM pode aplicar dimensionamento automático como solução de compatibilidade para aplicativos antigos que não foram desenvolvidos corretamente para DPI elevado.

22. Limitações nativas

O gerenciador padrão não oferece diretamente:

Regras avançadas permanentes de posicionamento por aplicativo.
Divisões completamente personalizadas da tela.
Fixar universalmente qualquer janela sempre no topo por um botão padrão.
Mover automaticamente programas para posições complexas.
Salvar qualquer layout personalizado com regras detalhadas.
Janelas em mosaico automático como alguns gerenciadores do Linux.
Controle completo das janelas de processos executados com permissões superiores.

Para ampliar essas funções, o Microsoft PowerToys oferece ferramentas como:

FancyZones: criação de zonas personalizadas.
Always on Top: manter uma janela acima das demais.
Workspaces: abrir e reposicionar conjuntos de aplicativos.
Utilitários adicionais de gerenciamento de monitores e janelas.
