18. Principais atalhos
Windows + E: abrir o Explorador.
Ctrl + C: copiar.
Ctrl + X: recortar.
Ctrl + V: colar.
Ctrl + A: selecionar tudo.
Ctrl + F ou Ctrl + E: pesquisar.
Ctrl + Shift + N: criar uma pasta.
F2: renomear.
Delete: enviar para a Lixeira.
Shift + Delete: excluir permanentemente.
Alt + Enter: abrir propriedades.
Alt + P: mostrar ou ocultar a visualização.
Alt + seta para cima: subir uma pasta.
Alt + seta para a esquerda: voltar.
Alt + seta para a direita: avançar.
Ctrl + N: abrir uma nova janela.
Ctrl + W: fechar a janela ou aba ativa.
Ctrl + Tab: trocar de aba no Windows 11.
F11: entrar ou sair da tela cheia.
